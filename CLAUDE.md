## gstack

Use `/browse` from gstack for all web browsing. Never use `mcp__claude-in-chrome__*` tools.

Available gstack skills:
`/office-hours`, `/plan-ceo-review`, `/plan-eng-review`, `/plan-design-review`,
`/design-consultation`, `/review`, `/ship`, `/land-and-deploy`, `/canary`,
`/benchmark`, `/browse`, `/qa`, `/qa-only`, `/design-review`,
`/setup-browser-cookies`, `/setup-deploy`, `/retro`, `/investigate`,
`/document-release`, `/codex`, `/careful`, `/freeze`, `/guard`, `/unfreeze`,
`/gstack-upgrade`.

If gstack skills are not working, run `cd .claude/skills/gstack && ./setup --host codex`
after installing Bun. On Windows, Node.js is also required for Playwright-backed browser flows.
